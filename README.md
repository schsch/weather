# weather
Simple js app for getting weather data from Yahoo

## Ready version

See [fornit1917.github.io/weather](http://fornit1917.github.io/weather)

## Build

Install node (version >= 4), npm (version >= 3), gulp and run:

```
npm install
gulp all
```

or, if you need js-optimizations:

```
npm install
gulp all-min
```
